搭建hexo博客(windows系统)

安装node.js （1-5）

1、node.js下载地址：https://nodejs.org/en/

2、安装过程一直next即可，此处我安装在D盘：

3、安装完成

4、接下来win+r打开cmd

5、查看node.js版本，输入node -v；查看npm版本，输入npm -v，看到版本号如图：


 

安装hexo（6-9）

6、此处为了提高速度，先利用npm安装一个淘宝的镜像语言cnpm。全局安装，输入npm install -g cnpm --registry=https://registry.npm.taobao.org，如图等待下载完成。

7、完成后，用cnpm -v验证一下，显示版本号

8、此时用cnpm正式全局安装hexo，输入cnpm install -g hexo-cli，

9、上图为安装完成，我们看下hexo版本，输入hexo -v，如下图

使用hexo搭建博客（10-19）

10、安装好hexo后，新建一个文件夹用来放博客内容，此处我建在D盘下，名为blog。后面如果出现错误，杀了它重建即可。

11、进入blog文件夹，右键点击Git Bush，输入hexo init，初始化一个博客，如下图请耐心等待。

12、如下图，则初步搭建成功了~点开blog文件夹，可以看到hexo生成的文件。

13、输入hexo s，启动博客，则会生成一个本地浏览页面，复制红框内容到浏览器地址栏，就可以看到新部署的网页了

14、网页是这样的：

15、关闭网页，然后Ctrl+C退出

16、下面我们新建一篇文章，输入hexo new “文章名”，例如下图，就生成了一篇新文章

17、可以看到，D:\blog\source_posts文件夹下出现一篇新文

18、此时我将已写好的文章内容复制进来 ，接着输入hexo clean；hexo g；hexo s，查看一下预览，如图

19、接下来，仍旧是关闭网页，Ctrl+C退出

将hexo博客部署到github上（20-26）

20、打开github主页，创建一个项目，用你的用户名.github.io命名

​ 具体可以看这篇：GitHub Pages 搭建教程

21、好的，你已经拥有一个自己的github仓库了，下面回到blog文件夹，右键git bush，输入cnpm install hexo-deployer-git --save，如图，安装git部署插件成功

22、下面设置config.yml文件，可使用记事本打开

23、拉到最下面，如图修改，注意空格

24、接下来将博客部署到远端，blog下右键git bush，输入hexo d，耐心等待。第一次部署要填github的账号密码，一般会自动弹出窗口。部署完成则如下图

25、此时再打开你的github仓库，会发现多了一些文件，如下图

26、浏览器地址栏输入yourname.github.io就会显示你的博客页。那么截至目前，博客搭建就完成了，后面的文章将介绍如何换主题、以及日常上传博文照片等。

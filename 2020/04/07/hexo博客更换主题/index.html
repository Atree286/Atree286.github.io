Hexo的next主题安装

​ 目前应用比较多两个主题是yilia和nexT，因为用这两个主题有改不明白的地方网上教程比较多，更容易上手，而且都很简洁漂亮。

使用yilia主题

1、文首视频里设置的是yilia主题，该项目地址是yilia，打开后 ，如图获取项目地址

2、打开blog文件夹，右键git bush，输入git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia，将其下载到theme文件夹下的名为yilia文件夹内。之后耐心等待clone完成。

​ 我这里clone时报错“ RPC failed; curl 56 OpenSSL SSL_read: Connection was reset, errno 10054”，查了一下原因有二，一是服务器的SSL证书没有经过第三方机构的签署，所以才报错，先输入env GIT_SSL_NO_VERIFY=true git clone https://<host_name/git/project.git，完成clone后再改回来，代码git config http.sslVerify "false"，参见链接；二是文件体积受限，输入 git config --global http.postBuffer 524288000 ，修改传输大小，参见链接。

​ 我这里使用第二种方法解决掉了，下载好yilia主题后显示如下图

3、下载好后，我们要更改主题，记事本打开blog根目录内的_config.yml文件

4、找到主题处，将其改为yilia，如图

5、顺便我们将名字也修改一下，如图

6、之后我们输入hexo clean；hexo g；hexo s，看一下效果，没有问题就可以，Ctrl+C退出，输入hexo d，上传GitHub就可以了

使用next主题

1、下载主题，在blog下右键git bush，输入 git clone https://github.com/iissnan/hexo-theme-next.git themes/next，完成后同yilia一样，修改blog下的_config.yml，将主题名改为next

2、改完三连：hexo clean；hexo g；hexo s，看看效果~然后hexo d上传，完成啦

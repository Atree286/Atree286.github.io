Git绑定Github（5-16）

5、到上步为止，我就没有管git，直接去安装node.js，然后进行到使用git命令hexo init时，报错“fatal: unable to access ‘https://github.com/hexojs/hexo-starter.git/': Failed to connect to github.com port 443: Timed out”，查了解决办法，原因有两个，①不该加代理不小心被代理软件加了代理。②需要FQ的的git地址没有加代理。这里我们的解决办法是设置URL为SSH，绑定github。

6、承接第5步，进入文件夹，右键git bush，输入git config --global http.proxy查询当前是否设置了代理

7、如下图显示我这里之前没有代理，于是输入git config --global --unset http.proxy取消代理

8、下面我们使用ssh将git绑定github。新建一个文件夹git codelife，此处我建在D盘下，如图

9、此文件夹下右键git bush，输入ssh查看一下是否已装ssh，下图显示已装

10、继续输入ssh-keygen -t rsa，然后敲四次回车，就会在C:\Users\阿波罗.ssh位置生成两个文件，分别是密钥id_rsa和公钥id_rsa.pub。

11、输入以下代码，使用Git Bush打开公钥。cd ~/.ssh；ls；cat id_rsa.pub，如图

12、使用Ctrl+Fn+PgDn组合键复制公钥，来到github主页，点击右上角settings，如图

13、然后点进SSH and GPG kays，点击New SSH key，如图

14、将你复制好的公钥内容粘贴，点击Add SSH key，如图

15、接下来会验证你的github密码，填写后即显示添加完成。验证一下是否添加成功，在git bush中输入ssh -T git@github.com

16、接着输入yes即可，出现下面文字，则绑定成功。

另：如果第15步输入后，git报错“Connection reset by 13.250.177.223 port 22”，可进入找到C:\Windows\System32\drivers\etc\hosts，记事本打开hosts文件，在最底部添加

192.30.255.112 github.com git
185.31.16.184 github.global.ssl.fastly.net

保存后，重新尝试第15步即解决。
